# 사용자 정보

## 개요

- **경로**: `/home/basic/users`
- **역할**: `om_user_m` 기반 사용자 목록 조회(등급 ADMIN).
- **API**: `GET /api/v1/admin/users?page=&size=&sort=` — 응답 `PagedData` (`docs/guide/02-개발-표준.md`).
- **UI**: OM과 동일한 `DataGrid.css`(헤더 가운데·셀 굵기 보정 포함) + `DataGrid` / `DataGridPaginationFooter`, `PageLayout`(공통코드와 동일: `h1.h4` 타이틀·`text-body-secondary small` 리드·`btn-phoenix-primary` 새로고침), `themeBalham` + `colorSchemeLightCold`, 다중 선택·Excel 받기. 하단 페이지 크기 옵션은 주문 목록과 같이 `100, 1000, 5000, 10000`.

## 연동

- 그리드에서 **사용자 ID** 클릭 시 `/home/admin/password-reset?userId=` 로 이동(비밀번호 초기화 화면에 ID 프리필).

## 권한

- 세션 등급 `ADMIN` 만 화면·API 사용.
